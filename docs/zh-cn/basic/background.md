## 背景渲染

> 包括图片背景和视频背景

| 参数名      | 类型   | 默认值 | 描述                 |
| ----------- | ------ | ------ | -------------------- |
| type        | String | image  | 背景类型             |
| link        | String | 空     | 背景图片（视频）链接 |
| extendClass | String | 空     | 扩展 css 类          |
